CodeForge 🚀

CodeForge is a full-stack coding platform where users can practice problems, write code, and track their progress.

✨ Features
Solve coding problems
Submit code and view results
Track solved problems
User authentication

🛠️ Tech Stack
Frontend: React, Vite
Backend: Node.js, Express
Database: PostgreSQL
ORM: Prisma

⚙️ Setup
1. Clone the repository  
2. Install dependencies  
3. Run the project  

```bash
git clone <repo-link>
cd CodeForge
npm install
npm run dev
